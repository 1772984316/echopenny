# EchoPenny

AI 女友陪伴 + 智能记账手机 App

跟 Echo 聊天，她陪你说话、记住一切；你说"吃饭12"，她笑着帮你记上。

## 技术栈

- Flutter 3.27+ / Riverpod
- DeepSeek API (Function Calling + 多模态)
- Drift (SQLite) 本地存储
- 用户自配 API Key，纯本地 App
